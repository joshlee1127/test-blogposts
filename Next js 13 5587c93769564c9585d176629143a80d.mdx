SSG = Static Side Generation vs Server Side Generation > Nextjs 可以 hybrid 這二個動作

SSG 對於 CND 運作來說是必須的

```jsx
.
├── README.md
├── app #page api components都在這
├── next-env.d.ts
├── next.config.js
├── node_modules
├── package-lock.json
├── package.json
├── postcss.config.js
├── public  # image等靜態資源
├── tailwind.config.ts
└── tsconfig.json
```

```jsx
joshlee@jump5:/mnt/c/Users/admin.JOSH/Desktop/myLab/next-js-course/next01/app$ tree -L 2
.
├── favicon.ico
├── globals.css
├── layout.tsx #lauout頁面
└── page.tsx #主頁面
```
